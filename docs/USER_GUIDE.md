# User Guide
## Venture Pipeline Management System (VPMS)

---

## 1. Introduction
This guide explains how to use the Venture Pipeline Management System (VPMS) to upload, manage, and review venture-related documents.

The system supports two primary user roles:
- Founders / Venture users
- MIV Admins

---

## 2. Accessing the System

### Logging In
1. Open the VPMS application URL.
2. Enter your registered email and password  
   *(or sign in using Google if enabled)*.
3. Click **Login** to access the dashboard.

After logging in, users are redirected to their dashboard based on their role.

---

## 3. Founder Guide

### Uploading Documents
1. Navigate to **Documents** from the dashboard menu.
2. Select the required **Document Type**.
3. Drag and drop your file or click **Browse Files**.
4. Upload supported file types (e.g. PDF, Word, Excel, PowerPoint).
5. Submit the document.

Once uploaded, the document status is automatically set to **Pending**.

---

### Viewing Uploaded Documents
1. Go to the **Documents** section.
2. View the list of documents you have uploaded.
3. Check the **Status** column for the current review state.

Possible statuses:
- **Pending** – Awaiting admin review
- **Approved** – Document accepted
- **Rejected** – Document rejected
- **Needs Changes** – Revisions required

---

### Updating or Replacing Documents
If a document is marked as **Needs Changes**, you may:
1. Review feedback (if provided).
2. Upload a revised version of the document.
3. Resubmit for review.

---

## 4. Admin Guide

### Reviewing Documents
1. Log in as an admin user.
2. Navigate to the **Documents** section.
3. View documents submitted across all ventures.
4. Open a document to review its content.

---

### Updating Document Status
1. Select a document.
2. Update its status to:
   - **Approved**
   - **Rejected**
   - **Needs Changes**
3. Save the status update.

The updated status becomes visible to the relevant founder.

---

## 5. Security & Access Rules
- Founders can only view documents associated with their own venture.
- Admins can view and manage documents across all ventures.
- File uploads are restricted by type and size.
- Documents can only be downloaded by authorised users.

---

## 6. Support
For access issues or system support, contact the MIV program team or system administrator.
